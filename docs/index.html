⚙ GitHub 連携設定
企業名の編集内容を直接GitHubのデータベースに保存するには、 GitHub Personal Access Token (PAT) が必要です。

PAT作成手順：
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. 「Generate new token (classic)」をクリック
3. スコープで repo にチェックを入れる
4. 生成されたトークンをコピーして下に貼り付ける
⚠ このトークンはお使いのブラウザのlocalStorageにのみ保存されます。 サーバーには送信されません。GitHubへの通信はHTTPS経由で行われます。 使用後は不要になれば削除できます。
